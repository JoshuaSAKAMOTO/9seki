xwOBA（Expected Weighted On-Base Average、期待wOBA）は、打球速度と打球角度、さらに一部の走塁状況（内野安打に効くスプリント性能）を入力に、各打席の期待 wOBA 値を算出する指標である。実際の結果ではなく「その打球が平均的にどれだけの得点価値を生むはずだったか」を示す。

wOBA 自体は、単打・二塁打・三塁打・本塁打・四球を得点価値で重み付けした総合打撃指標で、OPS よりも精緻に打撃貢献を評価する。xwOBA はそこから BABIP や HR/FB の運要素を取り除き、「打球の質」に基づく本質的な打力を抽出する。

## 数字の読み方

スケールは wOBA と共通で、リーグ平均は .315 前後。率指標のため、OBP や OPS と同じ感覚で読める。

- .400以上: 球界トップ級の打者（MVP候補水準）
- .370〜.399: エリート打者
- .340〜.369: 明確に平均以上
- .310〜.330: リーグ平均
- .290未満: 平均以下

シーズン通して .400 を維持できる打者は各年3〜5人程度で、MVP 投票と強く相関する。

## wOBA との差分の読み方

xwOBA の真価は、実測 wOBA との差分（wOBA - xwOBA）を見たときに現れる。差分が大きい選手は運に左右されており、翌シーズンに向けて xwOBA 側への収束が期待できる。

- **wOBA > xwOBA**: 実際の成績が期待値を上回る。BABIP や HR/FB の上振れを疑う
- **wOBA < xwOBA**: 実際の成績が期待値を下回る。不運の可能性が高く、反発余地がある
- **差分 ±.020 以内**: 実力と結果がほぼ一致

差分が .030 を超える場合、シーズン残りの期間で反動が起きる確率は高い。シーズン途中の打者評価では、wOBA より xwOBA を優先的に見るべきだ。

<Advanced>
xwOBA のモデルは打球速度と打球角度を入力とする GAM（一般化加法モデル）ベースで、Baseball Savant が公開している。内野安打に限って走塁速度も入力に加わる。一方、守備側のシフト状況や打球方向、球場要因は入力に含まれないため、極端なプルヒッター（シフトで抑えられやすい）や左打ちのゴロ打者は xwOBA を恒常的に下回る傾向がある。これを補正した指標として、FanGraphs の xwOBAcon（コンタクトのみ）や、打球方向を加味した Deserved Run Created などが存在する。
</Advanced>

## 使うときの注意

xwOBA は三振・四球・死球を実測値のまま使うため、打球の質だけでなく選球眼の影響も含む総合指標である。純粋に「打球の質」だけを比較したい場合は、xwOBAcon（コンタクト限定の xwOBA）を見る方が適切だ。

<Advanced>
xwOBA は In-Play の打球に対して算出されるため、内野ポップや高い打球角度のゴロのように、速度は出ていても明確に凡打となる打球も正しく低く評価される。つまり Hard-Hit% より悪い数字が出る打者もいる。ここが <Metric id="hard-hit-rate">Hard-Hit%</Metric> との決定的な違いで、xwOBA は「強さ × 角度 × 得点価値」の総合評価になっている。
</Advanced>
